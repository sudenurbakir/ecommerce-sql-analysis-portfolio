# E-Ticaret SQL Analiz Portfolyosu

## Proje Hakkında

Bu proje, SQL öğrenme sürecimi belgelemek ve öğrendiklerimi gerçek bir senaryo üzerinden uygulamak amacıyla hazırlanmıştır.

Senaryo olarak hayali bir e-ticaret şirketi seçilmiştir. Proje boyunca bir İş Analisti'nin karşılaşabileceği temel veri yapıları oluşturulacak ve bu veriler üzerinde SQL sorguları geliştirilecektir.

Bu proje tamamlandığında yalnızca SQL sorgularını değil, aynı zamanda öğrenme sürecimi de yansıtan bir portfolyo ortaya çıkarmayı hedefliyorum.

---

## Proje Hedefleri

- Veritabanı mantığını öğrenmek
- E-ticaret veri tabanı tasarlamak
- SQL sorguları yazmayı öğrenmek
- Gerçek iş problemlerini SQL ile çözmek
- Git ve GitHub kullanım pratiği kazanmak
